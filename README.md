\# MLOps Churn Prediction Project



This project demonstrates an end-to-end MLOps pipeline for predicting customer churn.



\## Tech Stack

\- Python

\- MLflow

\- DVC

\- FastAPI

\- Docker

\- Git



\## Pipeline

1\. Data versioning using DVC

2\. Model training with MLflow tracking

3\. Model artifact management

4\. FastAPI model serving

5\. Docker containerization



\## Run API

uvicorn api.main:app --reload
